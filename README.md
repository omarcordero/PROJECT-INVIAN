## Correr el Proyecto

El proyecto de Laravel no necesita ejecutar con un ningún script sql para que funcione, simplemente clonar el proyecto y ejecutar el comando:

- php artisan migrate

## Descripción del Proyecto

- Se usa migrations
- Ajax
- Servicios Rest
- Eloquent


## Modify .env

- DB_CONNECTION=mysql
- DB_HOST=127.0.0.1
- DB_PORT=3306
- DB_DATABASE=company_invian
- DB_USERNAME=root
- DB_PASSWORD=

## Run command

- php artisan migrate

## Run project

- php artisan serve
